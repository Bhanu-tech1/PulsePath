🚑 PulsePath – Saving Lives by Clearing the Path
> PulsePath is a smart emergency response platform that helps ambulances avoid potholes and damaged roads for faster response times. Citizens report road hazards with photos and locations, BBMP monitors and prioritizes repairs, and ambulance drivers receive real-time hazard-aware optimized routes — reducing delays and saving lives.

🚨 Problem Statement
Potholes and damaged roads delay ambulances during emergencies, increasing response time and putting lives at risk. Existing pothole reporting systems only focus on reporting road damage and do not assist emergency services in avoiding hazardous routes.

💡 Solution
PulsePath connects Citizens, Ambulance Drivers, and BBMP (Municipal Authorities) through a unified platform enabling:
- Real-time pothole reporting
- Route optimization for ambulances
- Road maintenance coordination

👥 User Roles
| Role | Responsibilities |
|------|-----------------|
| 🧑 Citizen | Report potholes with images, location & description |
| 🚑 Ambulance Driver | Get optimized, hazard-aware emergency routes |
| 🏛️ BBMP Authority | Monitor, prioritize, and track road repairs |

✅ Features
Core Features
- Multi-Role Login System – Separate dashboards for Citizens, Ambulance Drivers, and BBMP
- Pothole Reporting – Upload images, mark locations on map, add descriptions
- Interactive Map Integration – Visualize pothole locations and hazards
- Emergency Route Planning – Optimized routes avoiding pothole-prone roads
- GPS-Based Navigation – Route visualization, distance & travel time estimation
- BBMP Monitoring Portal – View, track, and prioritize road repairs
- Centralized Pothole Database – Stores location, images, severity, and timestamps

Advanced Features (Proposed)
- 🤖 AI-Based Pothole Detection – Computer vision to auto-detect and classify potholes
- 🚦 Real-Time Traffic Integration – Avoid congestion, closures & construction zones
- 📊 Smart Severity Classification – Auto-categorize potholes (Low / Medium / High / Critical)
- 🔧 BBMP Repair Tracking – Status updates: Reported → Inspected → Scheduled → Fixed
- 🚑 Ambulance Priority Routing – Fastest & safest route based on emergency level
- 📈 Analytics Dashboard – Insights on reported, fixed, and pending potholes
- 🔮 Predictive Road Maintenance – Predict pothole-prone areas using historical data
- 🏅 Citizen Reward System – Points, badges & recognition for active reporters
- 📡 Emergency Dispatch GPS Integration – Direct integration with ambulance dispatch systems

Getting Started

Prerequisites
- Node.js installed

Installation

```bash
1. Clone the repository
git clone https://github.com/Bhanu-tech1/PulsePath.git

2. Navigate to the project directory
cd PulsePath

3. Install dependencies
npm install

4. Set up environment variables
Create a .env.local file and add your API keys:
GEMINI_API_KEY=your_gemini_api_key_here

5. Run the development server
npm run dev
```
